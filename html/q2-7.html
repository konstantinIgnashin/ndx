Ну главное это теория, она дает сильный толчок в используемых технологиях, полное теоритическое понимание js можно почитать в книге Девида Фленагана "Руководство Javascript" тут собрано полное теоритическое изложение работы интерпритатора js, в конце книги есть справочник функций.
Для практики это в первую очередь javascript.ru, ну и топы гугла по нужным запросам, также много умников на пиарится на habrahabr.ru тут есть что почитать по методики, тесты, или просто рекомендации к разработкам. Также если вы используете jquery то на его сайте jquery.com есть полные описания работы всех функций этой библиотеки.
Также есть хорошие справочники по css2 и css3 htmlbook.ru, w3schools.com и др. (топы из гугла), там можно найти полные руководства по совместимости браузеров с css методами.
